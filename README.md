# 🔐 Password Based Door Lock System using Arduino

📌 Overview
The Password Based Door Lock System is an embedded security system designed using Arduino. It allows access only when the correct password is entered through a keypad, ensuring safety and controlled entry.

 🎯 Objectives
- Provide secure access using password authentication  
- Replace traditional lock systems with digital security  
- Improve safety for homes and offices  
- Create a user-friendly interface  

🧠 Working Principle
The system uses a 4x4 keypad to enter a password.  
- If the entered password is correct → Door opens  
- After a few seconds → Door closes automatically  
- If password is wrong → Error message is displayed  

The Arduino processes input and controls all components accordingly.

 🛠️ Components Used
- Arduino UNO  
- 4x4 Keypad  
- LCD Display  
- Servo Motor  
- Buzzer  
- Jumper Wires  

⚙️ Methodology
- User enters password using keypad  
- Arduino verifies the password  
- Servo motor controls door lock/unlock  
- LCD displays status messages  
- Buzzer alerts for incorrect attempts

 ⚠️ Limitations
